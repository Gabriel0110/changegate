# Third-Party Notices

ChangeGate includes a compact detection metadata snapshot derived from the public
[DataDog/pathfinding.cloud](https://github.com/DataDog/pathfinding.cloud) AWS IAM
privilege-escalation path catalog.

The embedded snapshot is limited to defensive detection metadata: path ID, name,
category, services, required IAM actions, references, and source path. ChangeGate
does not embed exploitation commands or run pathfinding.cloud at scan time.
